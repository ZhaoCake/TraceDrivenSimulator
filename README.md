# TraceDrivenSimulator

基于 Trace 驱动的 RISC-V 流水线性能模拟器。

## 目标

通过 Trace 驱动的模拟方式（指令轨迹输入而非完整指令集模拟），快速评估经典五级流水线在不同微架构策略下的性能表现——IPC、停顿分布、转发命中率等。

## 基本原理

输入是结构化 Trace 记录（每条指令的 PC、寄存器号、指令类型、访存地址、计算结果真值）。模拟器以这些"事实"为输入，建模五级流水线的数据流，而非建模硬件电路的计算过程——这是 Trace 驱动模拟的核心思想：**重放数据依赖，而非重放计算**。

## 现状

已完成经典五级流水线的建模，包含五阶段（IF/ID/EX/MEM/WB）、转发网络（EX/MEM → MEM/WB → 寄存器文件三级优先级）和 Load-Use 冒险停顿。15 个测试用例覆盖了基本流水线、转发链、停顿、优先级、x0 硬连线等场景。代码已拆分为 `src/simulator/` 子目录模块。
